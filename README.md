# Practice-3-IT
Задание выполняли Греков В.Д. и Буханцева Е.А. ; ГРУППА Т-11/1-25
структура html соответствует заданию. шапка сайта оформлена через header и содержит заголовок h1 и подзаголовок h2. основной блок main содержит контейнер gallery с тремя карточками flip-card. каждая карточка включает изображение, заголовок h3 с именем лошади, описание p и ссылку a на страницу wikipedia, открывающуюся в новой вкладке. каждая карточка уникальна: первая красная, вторая серая, третья фиолетовая, с плавным вращением при наведении или клике.

блок характеристик info содержит заголовок h2 и список ul из минимум четырех пунктов. дополнительно есть блок highlight для выделения текста и позиционирования абзаца с описанием символики лошадей. отдельный блок links содержит ссылки на страницы wikipedia с hover-эффектом. футер footer расположен в самом низу страницы и содержит текст о правах автора.

стили css включают общий сброс отступов, фон страницы черного цвета, шрифт arial, sans-serif, и красное неоновое свечение текста через text-shadow. карточки flip-card имеют ширину 250px, прозрачный фон, рамку и box-shadow для свечения, flip-эффект реализован через transform и transition. изображения имеют фиксированную ширину и высоту, object-fit: cover. ссылки оформлены без подчеркивания с изменением цвета при наведении. использованы минимум три класса (.flip-card, .info, .highlight) и типовые селекторы (h1, h2, ul, p, a).

каждая карточка имеет уникальный стиль. первая карточка красная с неоновым свечением, вторая серая, третья фиолетовая. три карточки выстроены в ряд через display: inline-block. flip-карточки интерактивны: javascript добавляет или убирает класс flip при клике.

javascript содержит следующий код:

const cards = document.querySelectorAll('.flip-card');

cards.forEach(card => {
  card.addEventListener('click', () => {
    card.classList.toggle('flip');
  });
});


код добавляет возможность переворачивать карточки при клике, создавая эффект flip.

проект полностью соответствует требованиям задания: шапка сайта с заголовком и подзаголовком, основной блок с карточками и ссылками, блок характеристик с минимум четырьмя пунктами, дополнительный блок для позиционирования текста, общие стили, стили карточек, фиксированные изображения, hover-эффекты ссылок, использование классов и типовых селекторов, три уникальные карточки в ряд с inline-block, интерактивность через javascript.
